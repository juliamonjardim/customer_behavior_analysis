# Customer Shopping Behavior Analysis

An end-to-end Data Analytics project analyzing customer shopping behavior using Python, PostgreSQL, SQL, and Power BI. This project simulates a real-world analytics workflow followed by data analysts in companies — from business problem definition and exploratory data analysis to database querying, dashboard development, reporting, and business recommendations.

The project analyzes 3,900 customer transactions across 18 features to uncover insights related to customer demographics, purchasing behavior, subscription trends, product performance, discount usage, and revenue contribution.

## Project Workflow

* Business Problem Definition
* Data Cleaning & Exploratory Data Analysis (Python)
* Data Modeling & Storage in PostgreSQL
* Advanced SQL Business Analysis
* Interactive Dashboard Development in Power BI
* Business Reporting & Presentation

## Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)

* SQL
* Power BI

* Gamma AI
* GitHub

## Key Insights

* Male customers generated ~68% of total revenue
* Young Adults contributed the highest revenue among age groups
* Loyal customers represented the majority of purchases
* Hats and Sneakers showed the highest discount dependency
* Subscription status showed strong influence on customer purchasing behavior
